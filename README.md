## 環境構築手順

### ソースをクローンする

### .env をコピーする

```bash
cp .devcontainer/.env.sample .devcontainer/.env
```

#### VsCode でプロジェクトフォルダーを開く

### Reopen in Container を押下

Ctrl Shift P → Reopen in container と入力して実行
